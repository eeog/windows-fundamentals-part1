# Settings vs Control Panel — System Configuration

Windows provides two interfaces for system configuration:

---

## ⚙️ Settings (Modern UI)

- User‑friendly  
- Touch‑optimized  
- Limited administrative depth  
- Used for basic configuration  

---

## 🖥️ Control Panel (Legacy)

- Deep system configuration  
- Required for many admin tasks  
- Still widely used in enterprise environments  

---

## 🛡️ Security Relevance

Analysts often inspect:

- Installed programs  
- Startup items  
- Network settings  
- User accounts  
- Security settings  

Attackers frequently modify these areas for persistence or evasion.
