# Task Manager — Process & Performance Analysis

Task Manager provides real‑time visibility into system activity, making it a critical tool for endpoint investigations.

---

## 📊 Key Tabs

### **Processes**
Shows running applications and background processes.

### **Performance**
Displays CPU, RAM, disk, and network usage.

### **Users**
Shows logged‑in users and their resource usage.

### **Startup**
Lists programs that run at boot.

### **Services**
Shows service status and allows quick control.

---

## 🔍 What Analysts Look For

- Unknown or suspicious processes  
- High CPU or RAM usage  
- Unexpected startup programs  
- Disabled security services  
- Multiple user sessions  

---

## 🛡️ Security Relevance

Task Manager is often the first place to spot:

- Malware  
- Persistence mechanisms  
- Resource abuse  
- Unauthorized activity
