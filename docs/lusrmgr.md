# Local Users & Groups (lusrmgr.msc)

Local Users and Groups is a Windows management console used to view and manage local accounts and groups.

---

## 🚀 Launching the Tool

Run:

```
lusrmgr.msc
```

---

## 👥 What You Can View

- All local user accounts  
- Group memberships  
- Account status (enabled/disabled)  
- Password requirements  
- Account descriptions  

---

## 🛠️ Common Administrative Actions

- Change account type  
- Reset passwords  
- Disable accounts  
- Add/remove group memberships  

---

## 🛡️ Security Relevance

Analysts use this tool to detect:

- Rogue admin accounts  
- Accounts created by attackers  
- Unexpected group memberships  
- Disabled accounts reactivated  

It’s a critical part of Windows endpoint investigations.
